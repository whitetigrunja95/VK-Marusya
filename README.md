# VK Marusya

Учебный frontend-проект — онлайн-кинотеатр с каталогом фильмов, поиском, страницами жанров и избранным.

## О проекте

Приложение позволяет просматривать подборки фильмов, искать фильмы, открывать подробную информацию по каждому фильму и добавлять фильмы в избранное.  
Интерфейс адаптирован под разные размеры экрана и сверстан по макету из Figma.

## Функциональность

- получение данных о фильмах с API
- отображение случайного фильма на главной странице
- вывод топа фильмов
- страница жанров
- страница фильмов по выбранному жанру
- поиск фильмов с выпадающим списком результатов
- страница отдельного фильма
- добавление и удаление фильмов из избранного
- хранение избранного в localStorage
- адаптивная вёрстка для desktop и mobile
- модальные окна для авторизации и трейлера
